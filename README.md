# Go-DS

Basicly, data structures package written with Go.  

Abstractly implemented data structures for now is given below. Any kind of feedback is appreciated.

- Stacks
- Queues
- Linked Lists
- Directed Graphs
- Undirected Graphs
- Binary Search Tree 
- Tries
- Hash Tables
- Heaps
